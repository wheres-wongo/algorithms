Searching: Ordered Arrays of Integers

Problem: Given an ordered array of elements, create an algorithm to tell if $x \in A'$

            
              def ordered_search(x: int, A: List[int]) -> bool:
              if A == []: 
                return False
              elif A[len(A) // 2] == x: 
                return True
              elif A[len(A) // 2] < x: 
                return ordered_search(x, A[len(A) // 2:]) 
              else: 
                return ordered_search(x, A[:len(A) // 2])
